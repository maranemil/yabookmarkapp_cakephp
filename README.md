# yabookmarkapp_cakephp
## Simple Bookmarks Web App  made with CakePHP 4.X

### Requirements

| Package | Version |
|---|---|
| PHP  | 7.4.16 (cli) |
| MySQL MariaDB | mysql  Ver 15.1 Distrib 10.5.12-MariaDB |
| Apache | Apache/2.4.46 (Ubuntu) |
| Bootstrap CSS | 4.0 |
| jQuery | 3.5.1 |
